# Software requirements.
- python 3.10^
- docker

# Usage python script.
## Create a virtual environment.
```bash
python -v venv pyenv
```
### For windows
```bash
/pyenv/Script/activate
```
## Run selenium stand alone with docker-compose.
```bash
cd /scraping_comments_from_group_fb
```
```
docker-compose up -d
```
## Run python script.
```bash
python main.py
```
